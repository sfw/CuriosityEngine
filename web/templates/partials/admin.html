{# 1. Re-run cross-reference #}

Re-run cross-reference

Runs cross-reference + synthesize + verify on the current journal state. Generates new xrefs only — the LLM sees existing xrefs as priors, the participant-set dedup + anti-attractor gate skip near-duplicates. Useful after a failed run, or when new entries have accumulated since the last cross-ref cycle.

Re-verify unregistered insights
